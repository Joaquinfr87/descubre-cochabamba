# **1. Descubre Cochabamba**

El presente proyecto busca desarrollar un **sitio web frontend de turismo** enfocado en el **departamento de Cochabamba, Bolivia**, con el objetivo de **promocionar sus atractivos turísticos, culturales y gastronómicos** de manera interactiva y visual.

Cochabamba es conocida como el “**Corazón de Bolivia**”, debido a su diversidad geográfica, riqueza cultural y gran oferta gastronómica. A través del sitio web, se pretende que los visitantes puedan conocer, planificar y disfrutar de la experiencia turística en la región de manera **intuitiva, atractiva y moderna**.

El proyecto se centra en el **frontend**, priorizando la **experiencia de usuario (UI/UX)**, la **accesibilidad** y la **responsividad** en distintos dispositivos, garantizando que el contenido se visualice correctamente en computadoras, tablets y smartphones. Se implementarán componentes visuales interactivos, como **mapas, galerías de imágenes, sliders de destinos, secciones de eventos y gastronomía**, todo organizado de manera intuitiva.

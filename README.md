# **1. Descubre Cochabamba**

El presente proyecto busca desarrollar un **sitio web frontend de turismo** enfocado en el **departamento de Cochabamba, Bolivia**, con el objetivo de **promocionar sus atractivos turísticos, culturales y gastronómicos** de manera interactiva y visual.

Cochabamba es conocida como el “**Corazón de Bolivia**”, debido a su diversidad geográfica, riqueza cultural y gran oferta gastronómica. A través del sitio web, se pretende que los visitantes puedan conocer, planificar y disfrutar de la experiencia turística en la región de manera **intuitiva, atractiva y moderna**.

El proyecto se centra en el **frontend**, priorizando la **experiencia de usuario (UI/UX)**, la **accesibilidad** y la **responsividad** en distintos dispositivos, garantizando que el contenido se visualice correctamente en computadoras, tablets y smartphones. Se implementarán componentes visuales interactivos, como **mapas, galerías de imágenes, sliders de destinos, secciones de eventos y gastronomía**, todo organizado de manera intuitiva.

# **Layout del Sitio Web – Descubre Cochabamba**


## **1. Logotipo**
![Logo de Descubre Cochabamba](public/logo.png)
**Concepto:**

* Representa el **corazón de Bolivia** y la diversidad de Cochabamba.
* Elementos:

  * Montañas (Cordillera de los Andes)
  * Sol o cielo despejado (clima y naturaleza)
  * Icono de comida típica (plato o utensilio)
* Estilo: **moderno, minimalista, con colores vivos**.
* Tipografía del logotipo: **amigable y legible**, similar a “Poppins” o “Montserrat”.

## **3. Paleta de Colores**

Inspirada en la **naturaleza y cultura de Cochabamba**:

| Color          | Código HEX | Uso sugerido                              |
| -------------- | ---------- | ----------------------------------------- |
| Verde Andino   | #2E7D32    | Botones principales, íconos               |
| Amarillo Sol   | #FBC02D    | Destacados, resaltados, hover             |
| Azul Cielo     | #0288D1    | Fondos secciones, enlaces                 |
| Naranja Tierra | #F57C00    | Llamadas a la acción, botones secundarios |
| Blanco         | #FFFFFF    | Fondo general y textos principales        |
| Gris Oscuro    | #424242    | Textos secundarios, footer                |

**Tip:** Mantener contraste alto para accesibilidad.

## **4. Tipografía**

Selección de tipografías **legibles y modernas**, con jerarquía clara:

| Elemento                   | Fuente sugerida | Estilo/uso               |
| -------------------------- | --------------- | ------------------------ |
| Títulos / Encabezados      | Montserrat      | Negrita, tamaños grandes |
| Subtítulos / Secciones     | Poppins         | Semi-bold                |
| Texto principal / párrafos | Roboto          | Regular, tamaño 16px     |
| Botones / CTA              | Montserrat      | Bold, mayúsculas         |
